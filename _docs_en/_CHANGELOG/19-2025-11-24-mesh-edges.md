### Detailed Mesh Editing
Added tools that allow refining the mesh structure and adding details.

### New Tools (Edit Mode)
- `mesh_bevel(offset, segments)`: Rounds off sharp edges. Operates on selected edges.
- `mesh_inset(thickness, depth)`: Creates new faces inside existing ones. Essential for paneling and hard-surface modeling.
- `mesh_loop_cut(number_cuts)`: Adds geometry resolution. Currently implemented as a subdivision wrapper for stability in headless environments.
